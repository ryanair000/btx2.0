# Premier League Match Prediction Web App

An AI-powered web application for predicting Premier League football match outcomes based on team statistics, recent form, and performance metrics.

## Features

- **Data-Driven Predictions**: Uses real match data to predict winners (Home Win / Away Win / Draw)
- **Confidence Metrics**: Displays confidence scores and accuracy estimates
- **Detailed Analysis**: Shows key factors influencing each prediction
- **Web-Based Interface**: Clean, modern UI for inputting fixtures and viewing predictions
- **RESTful API**: Submit multiple fixtures in JSON format for batch predictions

## Tech Stack

- **Frontend**: React 18, Next.js 15, TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes
- **Runtime**: Node.js

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation


pm install

### Development


pm run dev

Open [http://localhost:3000](http://localhost:3000) to view the app.

### Build

`
npm run build
npm start
`

## Notes

- Predictions are **informational only**, not guarantees
- The model requires accurate, up-to-date data for reliable predictions
- Missing data will lower accuracy estimates
- No betting advice or gambling terminology is used
